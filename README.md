### 進入ssh ###

* sudo-i :point_left:
* 安裝wget :point_left:
* 輸入一鍵腳本 :point_left:

腳本：wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubiBackup/doubi/master/ssrmu.sh && chmod +x ssrmu.sh && bash ssrmu.sh

* 可以自定義端口、用戶名、密碼 :point_left:
* 安裝完成後會顯示ssr鏈接以及二維碼 :point_left:
* 手機端配置即可 :point_left:
